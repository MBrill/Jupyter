Ein Modul für ebene Parameterkurven¶

Es liegt nahe alle ebenen Parameterkurven die wir in der Lehrveranstaltung behandeln in einem Modul zu sammeln. Dann knnen wir bei einer Aufgabe einen Kreis oder unser 'Standardbeispiel' einmal implementieren und wiederverwenden. Um dieses Notebook eigenstndig zu halten sehen wir hier davon ab und definieren einige Funktionen fr unsere Parameterkurven, die wir grafisch ausgeben.

Wir verwenden wenn mglich Keyword-Parameter. Als Rckgabe aller Funktionen geben wir zwei Felder NumPy-Felder zurck, fr die x- und fr die y-Koordinaten. Genau dieses Format erwarten beide Grafik-APIs.

import numpy as np


def first_example(a=-2.0, b=4.0, n=10):
    """
    Einstiegsbeispiel fr eine Parameterkurve in der Vorlesung.

    Parameter
    ----------
    a : float, optional
        Anfangswert des Parameterintervalls, Default ist -2.0.
    b : float, optional
        Endwert des Parameterintervalls, Default ist 4.0.
    n : int, optional
        Anzahl der Werte im Parameterintervall. Default ist 10.

    Returns
    -------
    x : float
        x-Werte der berechneten Punkte auf der Parameterkurve.
    y : float
        y-Werte der berechneten Punkte auf der Parameterkurve.
    """
    t = np.linspace(a, b, n)
    x = t*t-2.0*t
    y = t + 1.0
    return x, y


def circle(radius=1.0, a=0.0, b=1.0, n=4):
    """
    Kreis um den Ursprung als Parameterkurve.

    Parameter
    ----------
    radius : float, optional
        Radius des Kreises, Default ist 1.0.
    a : float, optional
        Anfangswert des Parameterintervalls, Default ist 0.0.
    b : float, optional
        Endwert des Parameterintervalls, Default ist 1.0.
    n : int, optional
        Anzahl der Werte im Parameterintervall. Default ist 4.

    Returns
    -------
    x : float
        x-Werte der berechneten Punkte auf der Parameterkurve.
    y : float
        x-Werte der berechneten Punkte auf der Parameterkurve.
    """
    t = np.linspace(a, b, n, endpoint=True)
    x = radius*np.cos(2.0*np.pi*t)
    y = radius*np.sin(2.0*np.pi*t)
    return x, y


def unit_circle(a=0.0, b=1.0, n=4):
    """
    Einheitskreis

    Parameter
    ----------
    a : float, optional
        Anfangswert des Parameterintervalls, Default ist 0.0.
    b : float, optional
        Endwert des Parameterintervalls, Default ist 1.0.
    n : int, optional
        Anzahl der Werte im Parameterintervall. Default ist 4.

    Returns
    -------
    x : float
        x-Werte der berechneten Punkte auf dem Einheitskreis.
    y : float
        y-Werte der berechneten Punkte auf dem Einheitskreis.
    """    
    return circle(radius=1.0, a=a, b=b, n=n)

Matplotlib¶

Wenn wir Matplotlib verwenden bentigen wir natrlich einen entsprechenden import. Dann erzeugen wir die Punkte auf der ebenen Parameterkurve, die wir darstellen mchten und verwenden diese als Eingabe in die Funktion plot. Dabei knnen wir steuern ob wir nur die Punkte oder einen Polygonzug darstellen mchten. In der Vorlesung haben wir mit den Punkten begonnen, die wir auch in einer Wewrtetabelle berechnet hatten.

In [25]:
import matplotlib.pyplot as plt

x, y = first_example(n=7)

fig = plt.figure()
plt.plot(x, y, 'go', markersize=3.0)
plt.title('Punkte der Parameterkurve $(t^2-2t, t+1)$')
plt.xlabel('x')
plt.ylabel('y')

plt.show()

Statt der Punkte einen Polygonzug auszugeben ist jetzt einfach. Wir verndern die Option fr die Funktion plt.plot. Sinnvoll ist es sicher auch mehr Punkte zu berechnen, damit wir den Eindruck einer Parameterkurve erhalten und nicht nur einen groben Polygonzug zu sehen. Wir erzeugen jetzt beide Abbildungen, einmal die Version mit 7 Punkten und dann mit einem deutlich feineren Polygonzug.

Die Zuweisung auf die Variable pio.renderers.default in diesen Notebooks sorgt dafr, dass man die Ausgabe in Jupyter Lab sieht. Die Notebooks werden in diesem Werkzeug verfasst. Damit die Ausgaben auch in den exportierten HTML-Dateien angezeigt werden, dafr sorgt der Renderer notebook_connected. Wir knnen bei Bedarf mit einem Plus-Zeichen noch weitere Renderer hinzufgen.
