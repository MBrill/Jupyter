Ebene Parameterkurven als Python-Funktionen

Wir implementieren einige Beispiele, angefangen mit unserem ersten Vorlesungs-Beispiel, das wir als first_example wiederfinden. Auch der Kreis als Funktion circle finden wir unten. Die Parameter wechseln abhngig von der Kurve. Beim Kreis mit Mittelpunkt im Ursprung ist es sicher sinnvoll, den Radius bergeben zu knnen. Was wir immer finden sind die Grenzen a und b fr das Parameterintervall und die Anzahl der Punkte im Polygonzug, den wir fr die Visualisierung einsetzen. Mit dieser Wert fr n erzeugen wir mit der Funktion np.linspace die t-Werte fr den Polgyonzug und berechnen anschlieend die x- und y-Werte, die wir mit return zurckgeben.

In[1]:
import numpy as np


def first_example(a=-2.0, b=4.0, n=10):
    t = np.linspace(a, b, n)
    x = t*t-2.0*t
    y = t + 1.0
    return x, y


def circle(radius=1.0, a=0.0, b=1.0, n=4):
    t = np.linspace(a, b, n)
    x = radius*np.cos(2.0*np.pi*t)
    y = radius*np.sin(2.0*np.pi*t)
    return x, y


def unit_circle(a=0.0, b=1.0, n=4):  
    return circle(radius=1.0, a=a, b=b, n=n)


def aufgabe2(a=0.1, b=5.0, n=4):
    t = np.linspace(a, b, n)
    x =t*t*t*t - t*t
    y = t + np.log(t)
    return x, y

Matplotlib

Wenn wir Matplotlib verwenden bentigen wir natrlich einen entsprechenden import. Dann erzeugen wir die Punkte auf der ebenen Parameterkurve, die wir darstellen mchten und verwenden diese als Eingabe in die Funktion plot. Dabei knnen wir steuern ob wir nur die Punkte oder einen Polygonzug darstellen mchten. In der Vorlesung haben wir mit den Punkten begonnen, die wir auch in einer Wertetabelle berechnet hatten.

In[2]:
import matplotlib.pyplot as plt

x, y = first_example(n=7)

fig = plt.figure()
plt.plot(x, y, 'go', markersize=3.0)
plt.title('Punkte der Parameterkurve $(t^2-2t, t+1)$')
plt.xlabel('x')
plt.ylabel('y')

plt.show()

Statt der Punkte einen Polygonzug auszugeben ist jetzt einfach. Wir verndern die Option fr die Funktion plt.plot. Sinnvoll ist es sicher auch mehr Punkte zu berechnen, damit wir den Eindruck einer Parameterkurve erhalten und nicht nur einen groben Polygonzug zu sehen. Wir erzeugen jetzt beide Abbildungen, einmal die Version mit 7 Punkten und dann mit einem deutlich feineren Polygonzug.

Plotly

Wenn wir Plotly verwenden ndert sich eigentlich nur der Teil, in dem wir eine Abbildung erzeugen und den Polygonzug ausgeben. Wir steigen direkt ein und erzeugen wieder eine Ausgabe unseres ersten Beispiels mit den 7 Punkten.

import plotly.graph_objects as go
import plotly.io as pio

pio.renderers.default = 'notebook_connected+jupyterlab'

Die Zuweisung auf die Variable pio.renderers.default in diesen Notebooks sorgt dafr, dass man die Ausgabe in Jupyter Lab sieht. Die Notebooks werden in diesem Werkzeug verfasst. Damit die Ausgaben auch in den exportierten HTML-Dateien angezeigt werden, dafr sorgt der Renderer notebook_connected. Wir knnen bei Bedarf mit einem Plus-Zeichen noch weitere Renderer hinzufgen.
