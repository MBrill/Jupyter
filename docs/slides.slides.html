Eine Präsentation¶

Wir können aus einem Jupyter Notebook eine Slide Show erstellen, die wir im Browser präsentieren. Dieses Beispielzeigt die bekannten Inhalte unseres Hello-World Notebooks als Folien.

Ein Beispiel mit Python Code¶

Wir importieren Numpy und geben den Wert von $\pi$ mit Hilfe von print aus.

In [4]:
import numpy as np
print('pi = ', np.pi)

pi =  3.141592653589793

Grafiken mit Matplotlib¶

Wir können Matplotlib dazu verwenden, um Grafiken im Notebook und in den Folien zu zeigen.

In [5]:
import matplotlib.pyplot as plt

plt.plot([1, 2, 4, 7])
Out[5]:
[<matplotlib.lines.Line2D at 0x286bf4d63d0>]
