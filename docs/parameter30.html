Aus der Lsung zum bungsblatt wissen wir, dass wir als Parameterintervall [-2, 2] verwenden. Wir implementieren eine Funktion mit den in der Aufgabenstellung angegebenen Funktionen f und g. Dabei importieren wir auch Numpy.

import numpy as np

def curvePoint(t):
    return t*(t*t - 2.0), t*(t-1.0)




def curve(a=-2.0, b=2.0, n=100):
    t = np.linspace(a, b, n)
    x = f(t)
    y = g(t)
    return x, y

Ausgabe von Tangenten und Normalen

Wir implementieren als erstes Funktionen, die den Geschwindigkeitsvektor, die Tangente oder die Normale fr einen Parameterwert berechnen. Da wir anschlieend die Funktion arrow in Matplotlib einsetzen geben diese Funktionen wie schon die Funktion curve x- und y-Werte zurck. Wir verwenden den Geschwindigkeitsvektor aus der Musterlsung. Fr die Tangente bringen wir diesen Vektor auf Lnge 1.

In[27]:
def speedVector(t):
    return 3.0*t*t - 2.0, 2.0*t - 1.0


def tangent(t):
    u = np.array(speedVector(t))
    # Wir erwarten eine regulre Kurve!
    return u/np.linalg.norm(u)


def normal(t):
    u = tangent(t)
    n = np.array([-u[1], u[0]])
    return n/np.linalg.norm(n)

Der Funktion arrow bergeben wir den Anfangspunkt der Vektoren, dafr haben wir die Funktion curvePoint implementiert. Und als drittes und viertes Argument bergeben wir die Richtung, die wir mit den beiden Funktionen tangent und normal berechnen. Wir geben erst einmal eine Tangente und eine Normale am Punkt zu Parameterwert t=1.0 aus.

Ausgabe mehrerer Vektoren

Wollen wir mehr als einen Vektor ausgeben bietet es sich an die Funktion quiver einzusetzen. Wenn wir dieses Wort bersetzen finden wir den sehr passenden deutschen Begriff Kcher. Dieser Funktion knnen wir ganze Arrays von Start und Endpunkten bergeben.

Die Funktion quiver skaliert die Vektoren um zu gewhrleisten, dass wir alle Vektoren erkennen knnen. Dazu definieren wir eine Variable quiverSacle. Bei der Interpretation der Grafik mssen wir das bercksichtigen, die Vektoren haben jetzt unterschiedliche Lngen, sie sind keine Einheitsvektoren mehr.

txy = tangent(tspecial)
print(txy.shape)

nxy = normal(tspecial)

quiverScale = 5

fig = plt.figure()
# Spur
plt.plot(x, y, 'b-')
# Punkt auf der Spur
plt.plot(xspecial, yspecial, 'ko', markersize=3.0)
# Tangenten
plt.quiver(xspecial, yspecial, 
           txy[0,:], txy[1,:],
           units='width',
           scale=quiverScale,
           pivot='tail',
           color='r')
# Normalen
plt.quiver(xspecial, yspecial, 
           nxy[0,:], nxy[1,:],
           units='width',
           scale=quiverScale,
           pivot='tail',
           color='g')
